How I install Arch Linux

Pre-Installation

Lang: zsh 
 1ping archlinux.org
 2
 3timedatectl
 4
 5cfdisk /dev/sdx
 6
 7mkfs.ext4 /dev/root_partition
 8mkswap /dev/swap_partition
 9mkfs.fat -F 32 /dev/efi_system_partition
10
11mkdir -p /mnt/boot/efi
12
13mount /dev/rootpartition /mnt
14mount /dev/efi_system_partition /mnt/boot/efi
15swapon /dev/swap_partition
